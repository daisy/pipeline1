A factory API for Fileset validators.

Discovery

The fact that a certain fileset can be represented as a {@link org.daisy.util.fileset.Fileset} does not mean that it automatically can be validated by a fileset Validator; implementations of the {@link org.daisy.util.fileset.validation.Validator} must be available on the system for the particular fileset type, and be registered so that a org.daisy.util.fileset.validation.ValidatorFactory can discover it.

Registering an implementation

The implementation discovery process is performed as follows:

  • First, System properties are checked for identifiers. The property string use syntax : org.daisy.util.fileset.validation:http://www.daisy.org/fileset/FILESETTYPE_STRING_CONSTANT, where FILESETTYPE_STRING_CONSTANT equals a toString constant in {@link org.daisy.util.fileset.FilesetType}.

    Example:

    System.setProperty( "org.daisy.util.fileset.validation:http://www.daisy.org/fileset/DTBOOK_DOCUMENT", "org.daisy.util.fileset.validation.ValidatorImplDtbook");
  • If no matching System property is found, a ValidatorNotSupportedException is thrown. (Discovery process is intended to be extended).

Note that a fileset instantiation in itself can be said to perform "basic" validation of a fileset: in XML filesets, wellformedness checking, DTD validation and URI resolvement (although not to the fragment level) is performed as an integral part of building the fileset. A Fileset Validator performs all the tests that go beyond the "basic" validation.

Usage example

Fileset mInputFileset = new FilesetImpl(inputURI,this,true,false); try{ //get a handle to the ValidatorFactory org.daisy.util.fileset.validation.ValidatorFactory validatorFactory = org.daisy.util.fileset.validation.ValidatorFactory.newInstance(); //try to produce a Validator for the particular Fileset type org.daisy.util.fileset.validation.Validator filesetValidator = validatorFactory.newValidator(mInputFileset.getFilesetType()); //register a ValidatorListener filesetValidator.setReportListener(this); //validate filesetValidator.validate(mInputFileset); }catch (ValidatorNotSupportedException e) { //the factory could not produce a validator for this fileset type }catch (ValidatorException ve) { //another error than nonsupported type occured }
