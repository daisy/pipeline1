This package constitutes the org.daisy.util extension implementations of the JAXP javax.xml.validation package.

The primary feature of this implementation is extension to support for RelaxNG, ISO Schematron and Schematron 1.5. Assuming that Xerces2.6 and JRE 1.5 or later are installed on the system, this means that usage of the javax.xml.validation package for XSD, RNG and SCH is possible in an implementation anonymous, opaque and orthogonal way.

The JRE shipped {@link javax.xml.validation.SchemaFactoryFinder} is used to locate the factories, still the following system properties must be set in order for the implementations in this package to be discovered:

System.setProperty( "javax.xml.validation.SchemaFactory:http://relaxng.org/ns/structure/1.0", "org.daisy.util.xml.validation.jaxp.RelaxNGSchemaFactory"); System.setProperty( "javax.xml.validation.SchemaFactory:http://www.ascc.net/xml/schematron", "org.daisy.util.xml.validation.jaxp.SchematronSchemaFactory"); System.setProperty( "javax.xml.validation.SchemaFactory:http://purl.oclc.org/dsdl/schematron", "org.daisy.util.xml.validation.jaxp.ISOSchematronSchemaFactory");

Usage Example:

javax.xml.validation.SchemaFactory rngFactory = SchemaFactory.newInstance(SchemaLanguageConstants.RELAXNG_NS_URI); rngFactory.setErrorHandler(this); rngFactory.setResourceResolver(CatalogEntityResolver.getInstance()); javax.xml.validation.Schema s = rngFactory.newSchema(new File("D:\\dtbook-2005-1.rng")); javax.xml.validation.Validator v = s.newValidator(); //validator is reusable, given that the schema is the same: File[] files = new File[] {new File("D:\\doc.dtbook"),new File("D:\\doc2.dtbook")}; for (int i = 0; i < files.length; i++) { v.validate(new StreamSource(files[i])); }

Constants for the SchemaFactory newInstance() call are avaialable in {@link org.daisy.util.xml.validation.SchemaLanguageConstants}.
