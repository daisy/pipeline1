Peek into an XML document prolog and head.

Peeking is the activity of, in a cost-minimized way, retrieving XML document prolog and head information.

Use PeekerPool to retrieve Peeker implementations.

For maximized system efficiency, always release (return) the peeker to the pool when done.

Usage example:

Peeker peeker = PeekerPool.getInstance().acquire(); try{ PeekResult result = peeker.peek(url); String rootNamespace = result.getRootElementNsUri(); }catch (Exception e) { //wasnt an XMLFile or something else went wrong }finally{ PeekerPool.getInstance().release(peeker); }
