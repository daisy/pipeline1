Introduction

The DMFC is an open source framework for document and DTB-related transformations. The DMFC may be downloaded from www.sourceforge.net, compiled, and run from the command line. The DMFC GUI and Windows installer are reserved for members of the DAISY Consortium. A Windows installer for the DMFC and GUI are available from the download pages of the DAISY Consortium web site, www.daisy.org/download/.

Definitions

  • Conversion - Change a file from one type to another. For example, change a file from a WordML file to an XHTML document, or an RTF document to a DTBook 2005 file.
  • Transformers - Used in conversions to change a file from one file type to another. A conversion may use several transformers to complete the change. For example, the Narrator (z3986 2005-1) conversion, changes a DTBook 2005-1 file to a complete DAISY/NISO z3986-2005 file set. Several transformers are "chained" together, including Speech Generators, Validators, and File Set Creator transformers to complete the conversion.
  • Job - A job consists of a Conversion, the file (source), and the folder or file destination.

Getting Started

The DMFC, DMFC GUI, and the Windows installer are available to members of DAISY through the DAISY web site.

  • Go to the DAISY web site, www.daisy.org/.
  • Login as a member.
  • From the main/top menu, choose Tools.
  • From the side navigation menu, choose Production Tools. A list of available tools will appear.
  • From the list of available tools, choose the DAISY Multi-Format Converter and Installer.
  • Download the installer as a zip file.
  • Unzip the installation file
  • Run the set-up file. The DAISY Multi-Format Converter will appear in the Start/All Programs/DMFC GUI shortcut menu.

Using the DMFC GUI If You Have A Disability

The DMFC GUI is written with the Standard Widget Toolkit (SWT), and is accessible to screen readers out of the box. The GUI may be navigated with a mouse or through the top, main menu. Standard Windows shortcuts are available in the DMFC GUI and additional shortcuts are defined in the main menus.

Choose a Conversion

A list of all Conversions by category are shown in the Select Converter box. Categories are determined by the type of source file. For example, in the "RTF" category, there are two conversions, "RTF to DTBook" and "RTF to XHTML". You must select a Conversion first.

  • Choose a Conversion with the Mouse

    The Conversions are listed in a tree structure by category. As each converter is highlighted, a description of each Converter appears in the Converter Description window. Highlighting the conversion selects it, there is no need to use the "Enter" key.

  • Choose a Conversion with the Menu

    From the top menu, choose Action/Select Converter. The focus is placed in the Select Converter window, and each conversion may be accessed by expanding the category and arrowing through the conversion choices. The Conversions are listed in a tree structure by category. As each converter is selected, a description of each Converter is in the Converter Description window, one tab over.There is no need to use the "Enter" key to make a Converter Selection.

Choose a Single File or Source to Convert

Use this option to change a single file (source) to another file type. First, select a Conversion. Choose the "Add Single Source" button. Alternatively, from the top menu, choose Action/Add Single Source. The Add Single Source dialog appears.

Add Single Source Dialog

Use this dialog to select a source file to convert and the output file or folder destination.

  • The "Name of the Converter" is automatically filled in from the converter chosen in the main screen.

  • Choose the first "Browse" button in the Source area to select a single file to convert. The "Choose Input File" dialog appears, it is a standard windows File dialog box. Alternatively, from the main menu, choose Action/Select Single Source.

    Only files that are compatible with the conversion appear in the Choose Input File dialog. For example, if the RTF to XHTML conversion is chosen, only .rtf files are shown in the dialog.

    The source file must be chosen from the dialog. The source text box cannot be edited.

  • Choose the second "Browse" button in the Destination area to select a destination file, or destination folder. Alternatively, from the main menu, choose Action/Select Destination.

    Depending on the conversion chosen, either a File or Folder dialog will appear. For example, the Narrator Conversion requires a single input (source) file and a destination folder for the fileset. The RTF to XHTML conversion requires a single input (source) file and a destination file.

    The destination file or folder must be chosen from the dialog. The destination text box cannot be edited.

  • To save the conversion, source and destination, choose the "OK"button, or from the main menu, Choose Action/OK-Save. The information appears in the main window in the Conversion Jobs table.

  • To exit without saving the conversion, source, and destination, choose the "Cancel"button, or from the main menu, Choose Action/Cancel.

Conversion Jobs Table

The Conversion Jobs Table lists Jobs in the order of Job selection from either the Add Single Source or Add Multiple Sources dialog. A job consists of a Conversion, the file (source), and the folder or file destination. Jobs are completed (run) top to bottom, in the order that they appear in the table. The Job may have a status of Waiting, In Progress, Completed, or Failed.

Modify the Conversion Job List

Modify the Conversion Job list to change the order of Jobs, or the composition of the list.

  • Move a Job Up in the List

    From the Conversion Jobs table, select/highlight a job. Choose the "Move Up" button. The job will move up, or one position closer to the top of the list.

  • Move a Job Down in the List

    From the Conversion Jobs table, select/highlight a job. Choose the "Move Down" button. The job will move down, or one position closer to the bottom of the list.

  • Clear or Remove Job(s) From the List

    From the Conversion Jobs table, select/highlight one or more jobs. Several jobs may be selected by using the shift or control keys. Choose the "Remove Job" button. The job will be deleted from the job list.

  • Change or Edit a Single Job

    From the Conversion Jobs table, select/highlight a job. Choose the "Change Job" button. The job Edit Conversion dialog will appear. The user may change the source or destination of the job, but not the conversion. Save changes with the "OK" button, and the changed job will appear at the bottom of the Conversion Job List.

    To ignore changes to a job, choose the "Cancel" button. The original job will appear at the bottom of the Conversion Job List.

Running the Conversion

To start the converters, choose the "Run"button. The jobs will be run from the top to the bottom of the job list.

  • Conversion Status

    Every job starts with a status of "Waiting" As the job is run, the status in the Conversion Jobs table changes to "In Progress", and finally to "Completed", if successfully run, or "Failed- see Log" if the run was not successful.

    To view the log, from the main menu, choose LogFile/View Logfile. If the run was not successful, copy the text in the dialog, and send the copy to your system administrator.

  • Transformers in Conversion

    Many conversions (such as Narrator) consists of many transformations. The "Transformers in Conversion" table lists all transformers in the conversion, and checks off each transformer as the transformer completes in the conversion.

    The total amount of time for each transformer is reflected in the "Total Expected Transformer Time", the amount of time left for completion of the transformer is reflected in "Remaining Transformer Time" and the progress of the transformer is shown in the "Transformer Progress" bar. Note that all these relate to the each individual transformer in the conversion, and not the total of all transformers in the conversion.

  • Cancel Running All Conversions

    To cancel all conversions, choose the "Cancel All Jobs" button.

Clear Finished Jobs from the Conversion Job List

To remove all jobs with a status of "Completed" or "Failed" from the Conversion Jobs table, choose the Clear Finished Jobs button. All jobs with a status of "Completed" or "Failed" will be cleared/removed from the list.
